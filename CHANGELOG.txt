# CHANGELOG - LudusaviPlaynite Extension

## Version 0.19.2 (2026-01-21)

### Added
- Sistema di logging personalizzato con file dedicati per giorno
- Rotazione automatica log (mantiene 30 giorni)
- Supporto automatico per giochi RPCS3/PS3
- Rilevamento automatico codici gioco PS3 (BLES, BLUS, NPEA, etc.)
- Ricerca dinamica cartelle savedata RPCS3 con match parziale
- Setting per percorso RPCS3 personalizzato
- Traduzioni italiano/inglese per nuove impostazioni

### Changed
- Migliorati messaggi log exit codes con descrizioni dettagliate
- Campo alias rimosso completamente dai custom games (invece di stringa vuota)
- Percorsi convertiti a forward slash per compatibilità YAML
- Logging più dettagliato per operazioni emulate
- Gestione errori più robusta per verifica esistenza custom games

### Fixed
- Risolto problema riconfigurazione ripetuta che corrompeva config.yaml
- Risolto exit code 101 per custom games già configurati
- Controllo esistenza custom game prima di usare configurazione cached
- Ricreazione automatica custom game se eliminato da Ludusavi
- Risolto problema percorso log (ora usa GUID invece di nome plugin)
- Gestione corretta backslash in percorsi Windows
- Prevenzione corruzione config.yaml durante verifica esistenza custom game
- In caso di errore durante verifica, assume che custom game esista per evitare corruzione

### Technical
- Nuova classe CustomLogger per gestione log dedicati
- Nuova classe RPCS3SaveFinder per rilevamento automatico salvataggi PS3
- Metodo CustomGameExists con gestione errori migliorata e logging dettagliato
- Try-catch in TryAutoConfigureEmulatedGame per prevenire corruzione configurazione
- Controllo AlternativeTitle prima di riconfigurare giochi emulati
- Script helper PowerShell/Batch per accesso rapido ai log
